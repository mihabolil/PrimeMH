# d2r-jbmh

## Run from the executable

Edit your `settings.toml` file, you need to set the `d2lodpath` which is just a copy of D2 LoD v1.13c, the same version used by the mapserver.

Everything else you can leave as defaults.

## Run from source

- Install Rust <https://www.rust-lang.org/tools/install>
- Download the source code to a folder on your PC <https://github.com/joffreybesos/d2r-jbmh/archive/refs/heads/master.zip>
- Open that folder in cmd prompt
- `cargo run --release`

I recommend installing VSCode, opening the folder with the code, then opening a terminal (cmd) session from there.
